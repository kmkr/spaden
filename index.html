Layout

Most pages on finn.no runs on the same .container > .page structure, these give the current widths to fit within the banners, and also the correct margins and responsive behaviour.

Basic template 
<!DOCTYPE html>
<html lang="no" prefix="og: http://ogp.me/ns#">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <meta http-equiv="X-UA-Compatible" content="IE=Edge">
        <title>FINN</title>
        <link rel="stylesheet" href="//static.finncdn.no/bb/latest/css/spaden/5.x/spaden.min.css">
        <!--[if IE 9]>
        <link rel="stylesheet" href="/static/ie9.e7628c1e.css">
        <![endif]-->
    </head>
    <body>
        <div class="container">
            <div class="page">
                <!-- page content -->
                <div class="line">
                    <div class="unit r-size2of3">
                        ...
                    </div>
                    <div class="unit r-size1of3">
                        ...
                    </div>
                </div>
            </div>
        </div>
    </body>
</html>

You can start with this basic HTML template.

Grid system for all FINN pages

line 
<div class="line">
    ...
</div>

Groups units on one horizontal line. Note: for mobile layout units may be stacked to avoid horizontal scrolling.

unit 
<div class="line">
    <div class="unit">
        ...
    </div>
</div>

Base class which divides a line into sections (columns).

r-sizeXofY 
<div class="line">
    <div class="unit r-sizeXofY">
        ...
    </div>
</div>

Extends unit. Indicates the number of grid units this element spans, for example r-size2of3 would take up 2 of 3 columns. Collapses to 1 column at 768px.

Intended use: Main layout grid!

sizeXofY 
<div class="line">
    <div class="unit sizeXofY">
        ...
    </div>
</div>

Supported fractions:

  • 1
  • 1/2
  • 1/3
  • 2/3
  • 1/4
  • 3/4
  • 1/5
  • 2/5
  • 3/5
  • 4/5
  • 1/6
  • 5/6

Extends unit. Indicates the fractional width of the unit, for example size3of4 would take up three quarters, or 75%, of the horizontal space.

Intended use: Non-responsive liquid elements

r-colsXtoY 
<div class="line r-colsXtoY">
    <div class="unit">
        ...
    </div>
</div>
<form>
  <div class="line r-cols1to2">
    <div class="unit formelement man">
      <label>
        <input type="checkbox">
        <span>Akershus</span>
      </label>
    </div>
    <!-- etc -->
  </div>
<form>

.r-cols1to2 .r-cols1to3 .r-cols1to4 .r-cols2to3 .r-cols2to4 .r-cols2to5 .r-cols2to6

Extends line. Responsive column setup for content. Classname indicates number of colums, for example r-cols2to4 will fit 2 columns on mobile screens and 4 columns from 768px and up.

Intended use: Column based contents

colsXupto[Breakpoint] 
<div class="line cols2upto480 cols4upto768 cols6upto990 cols8from990">

    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
    <div class="unit"> ... </div>
</div>

Supported column count: 1 - 8

Supported breakpoints:

  • 480
  • 640
  • 768
  • 990

Extends line.

.cols[X]upto[breakpoint] Sets number of columns up to specified breakpoint

.cols[X]from990 Sets number of columns from 990 and up

Classname indicates number of colums at specific breakpoints, for example .cols1upto480 .cols2upto768 .cols4from990 will fit 1 column below 480px, 2 columns between 480px and 767px, and 4 columns from 990px to infinity

! Works in conjunction with colsXfrom990, remember to set this aswell, or 990px+ will get whatever is default (usually 1col )

Intended use: Column based contents where we need more control

Responsive helpers

lt = less than
ht = higher than

Text alignment

.leftify-ht768 Adjust left over 768px
.leftify-lt768 Adjust left under 768px
.rightify-ht768 Adjust right over 768px
.rightify-lt768 Adjust right under 768px

Floating

.fleft-ht768 Float left over 768px
.fleft-lt768 Float left under 768px
.fright-ht768 Float right over 768px
.fright-lt768 Float right under 768px
