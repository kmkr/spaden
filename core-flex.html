Flex: Properties that may be set directly on specific flex-items

flex-grow[n] 
<div class="flex align-items-center">
    <div class="flex-grow1"> ... </div>
    <div class="flex-grow2"> ... </div>
    <div class="flex-grow1"> ... </div>
</div>

Defines the flex-item's ability to grow, Sets how 'free' space is distributed along the flex cross-axis (behaves Like justify-content but the coss direction from the specified flex-direction )

Supported values: 1 - 10 (Default:0)

flex-no-shrink 
<div class="flex align-items-center">
    <div> ... </div>
    <div> ... </div>
    <div class="flex-no-shrink"> ... </div>
</div>

This defines the ability for a flex item to shrink if necessary.
