Image formatting

Standard 4:3 image format 
<div class="format4by3">
    <img class="centered-image" src="dist/img/dummies/spade-510x384.jpg">
</div>

By default the image fills either the full width or height of the container without cropping, centered both horizontally and vertically.

Aspect ratios

Widescreen 16:9 image format 
<div class="format16by9">
    <img class="centered-image" src="dist/img/dummies/spade-510x287.jpg">
</div>

Sets the relative width and height of the image container.

Ratio Description
.format1by1 Square format (equal sides).
.format4by3 Standard mobile and digital camera format.
.format3by2 Professional camera format.
.format16by9 Widescreen format intended for images that span the full width of the page.
.format2by1 Even wider format.

Responsive helpers

16:9 on desktop, 3:2 on mobile 
<div class="format16by9 format3by2-lt768">
    <img class="centered-image" src="//placekitten.com/510/290">
</div>

Applies image formats depending on screen size.

lt = less than

.format1by1-lt768 Format 1:1 under 768px
.format4by3-lt768 Format 4:3 under 768px
.format3by2-lt768 Format 3:2 under 768px
.format16by9-lt768 Format 16:9 under 768px
.format2by1-lt768 Format 2:1 under 768px

Cropping

Horizontal cropping 
<div class="format4by3">
    <img class="centered-image crop-horizontal" src="dist/img/dummies/spade-510x287.jpg">
</div>

Cropping applied to an image possibly wider than its container, filling the height and cropped on the sides.

Vertical cropping 
<div class="format4by3">
    <img class="centered-image" src="dist/img/dummies/spade-510x384.jpg">
</div>

Cropping applied to an image possibly higher than its container, filling the width and cropped top and bottom.
